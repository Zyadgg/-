# نظام إدارة عروض الأسعار

نظام بسيط لإدارة وعرض عروض الأسعار مع واجهة إدارية لتعديل البيانات مباشرة.

## كيفية الاستخدام

### 1. تشغيل الخادم

افتح Terminal/PowerShell في مجلد المشروع وقم بتشغيل:

```bash
node server.js
```

أو:

```bash
npm start
```

### 2. الوصول للصفحات

- **عرض السعر**: افتح `http://localhost:3000/index.html`
- **صفحة الإدارة**: افتح `http://localhost:3000/admin.html`

### 3. تعديل البيانات

1. افتح صفحة الإدارة (`admin.html`)
2. قم بتعديل البيانات (العنوان، العميل، المجموعات، العناصر)
3. اضغط على "حفظ البيانات" لتحديث ملف `data.json` مباشرة
4. افتح صفحة عرض السعر لرؤية التحديثات

## الملفات

- `index.html` - صفحة عرض السعر
- `admin.html` - صفحة إدارة البيانات
- `data.json` - ملف البيانات
- `server.js` - الخادم البسيط
- `main.js` - كود عرض السعر
- `admin.js` - كود صفحة الإدارة

## ملاحظات

- تأكد من تشغيل `server.js` قبل استخدام صفحة الإدارة
- البيانات تُحفظ مباشرة في ملف `data.json`
- الإجمالي يُحسب تلقائيًا من مجموع أسعار العناصر
